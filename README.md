CineIFMA

Este é o projeto CineIFMA, uma plataforma desenvolvida para um estudo e testes ultilizando docker.
📋 Pré-requisitos

Antes de começar, certifique-se de ter instalado em sua máquina:

    Git

    Docker e Docker Compose

    Node

🚀 Como Executar (Docker)

A forma mais rápida de rodar o projeto é utilizando o Docker, que já configura todo o ambiente necessário.

    Clone o repositório:

    git clone https://github.com/Matheus-Sill/CineIFMA.git
    cd CineIFMA

    Inicie o projeto:

    docker-compose up --build

    Acesse a aplicação:
    Abra seu navegador e acesse http://localhost:3000
